# Port information

**Port 2**: The right wheel motors
**Port 3**: The left wheel motors
**Port 4**: Left arm
**Port 5**: Right arm
**Port 6**: Left fork
**Port 7**: Right fork
**Port 8**: Left claw
**Port 9**: Right claw

All the motor setup should be done on the program. If nothing happens, resend to the brain

# Controls

Left Joystick: move
Right Joystick: arms
Right triggers: fork
Left triggers: claw
